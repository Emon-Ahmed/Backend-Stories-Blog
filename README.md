# Minimal Tech Blog (Frontend)

Simple blog app frontend built with React and Vite.

## What it is

- Home page with posts list
- Pages for categories, authors, blog details
- Login and signup forms
- API calls via Redux Toolkit

## Run locally

1. `cd Backend-Stories-Blog`
2. `npm install`
3. `npm run dev`
4. Open `http://localhost:5173`

## Notes

- The backend API is in `Backend-Stories-Blog-Server`
- Change API URL in `src/features/api/apiSlice.js` if needed
